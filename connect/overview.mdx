spark ingests data through integrations, webhooks, and API calls. this section covers how to connect your tools and how to query spark programmatically.

## three ways to connect

<CardGroup cols={3}>
  <Card title="integrations" icon="cable">
    native connectors for github, slack, asana, linear, and shopify. events flow in automatically.
  </Card>
  <Card title="API & webhooks" icon="webhook">
    custom event ingestion and full read access to all spark data over HTTP.
  </Card>
  <Card title="MCP server" icon="cpu">
    AI agents query spark as native tools. works with claude desktop, cursor, and openclaw.
  </Card>
</CardGroup>
